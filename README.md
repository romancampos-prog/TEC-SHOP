# TEC-SHOP
Proyecto escolar de tienda en línea con arquitectura cliente-servidor.

## Tecnologías
- Frontend: React
- Backend: Node.js + Express
- Base de datos: MySQL (AWS RDS) (responsable: equipo BD)
- Imágenes: Firebase Storage
- Hosting: Front (Firebase Hosting o Render) / Back (Render)

## Estructura del repositorio
- `/frontend` → Aplicación React
- `/backend` → API REST (Node + Express)

## Reglas rápidas del equipo
- El equipo de frontend solo modifica `/frontend`
- El equipo de backend solo modifica `/backend`
- La base de datos la trabajan todos
