## Funcionalidades del sistema TEC-SHOP

### Usuarios
- Registro de usuarios
- Inicio de sesión
- Edición de perfil básico

### Productos
- Publicar productos con título, descripción, imagen y categoría
- Editar productos propios
- Eliminar productos propios
- Listar productos

### Imágenes
- Subida de imágenes a Firebase Storage
- Visualización de imágenes

### Generales
- Autenticación mediante JWT
- Comunicación vía API REST (JSON)

### Funcionalidades futuras
- Chat entre usuarios
- Sistema de calificaciones
- Métodos de pago informativos
